### Instructions

- `$ composer require slim/slim "^3.0"`
- `$ touch .gitignore`
- `$ mkdir -p src/public/`
- `$ cd src/public/`
- `$ vim info.php` just look over the file
- `$ vim index.php` just look over the file
- `$ cd ../..`
- `$ docker-compose up -d`
